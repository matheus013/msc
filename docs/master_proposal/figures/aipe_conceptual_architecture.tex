\caption{Arquitetura conceitual do framework AIPE. O framework separa a camada adaptável à base, responsável por padronizar dados transacionais e parâmetros operacionais, do núcleo metodológico reutilizável, composto pela caracterização da demanda, benchmark de políticas, simulação, validação estatística, geração de rótulos e treinamento do \textit{Policy Selection Engine}. Essa separação permite aplicar o AIPE a outras bases varejistas mediante adaptação da camada de entrada, preservando o núcleo de seleção contextual de políticas. A camada de aplicação utiliza o PSE para recomendar políticas a novas séries loja-produto e reavaliar periodicamente as recomendações conforme novos dados se acumulam.}
