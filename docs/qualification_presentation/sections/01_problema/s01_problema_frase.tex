% Slide 2 — Problema em uma frase
\begin{frame}{O problema em uma frase}
  \begin{center}
    \vspace{1.5em}
    {\Large\textcolor{darkblue}{\textbf{Redes varejistas heterogêneas precisam decidir}}}\\[0.6em]
    {\LARGE \highlight{quando} e \highlight{quanto} repor estoque}\\[0.6em]
    {\Large\textcolor{darkblue}{\textbf{em cada série loja-produto.}}}
    \vspace{1.5em}
  \end{center}
  \note{
    Essa é a frase que resume o ponto de partida da dissertação. Toda loja,
    para todo produto, enfrenta repetidamente essa mesma decisão: quando
    emitir um pedido e em qual quantidade. Parece simples, mas em uma rede
    com milhares de séries loja-produto, e com padrões de demanda muito
    diferentes entre elas, essa decisão repetida é o cerne do problema de
    controle de inventário que motiva todo o restante do trabalho.
  }
\end{frame}
