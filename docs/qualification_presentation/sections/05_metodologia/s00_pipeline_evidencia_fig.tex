\begin{frame}{Síntese do pipeline metodológico}
  \centering
  \includegraphics[width=\linewidth,height=6.0cm,keepaspectratio]{figures/aipe_evidence_pipeline.pdf}\\[0.3em]
  {\footnotesize\textcolor{slategray}{Mapa da metodologia: dados $\rightarrow$ características $\rightarrow$ simulação $\rightarrow$ perfis $\rightarrow$ rótulos $\rightarrow$ PSE.}}
  \note{
    Esta figura sintetiza a metodologia que será detalhada nos próximos
    slides. A leitura é top-down: primeiro a unidade de análise é preparada;
    depois a série loja-produto é descrita por características e sinais
    auxiliares; em seguida o simulador avalia todas as políticas candidatas
    nas mesmas condições; os regimes e PODs organizam o contexto
    operacional; o rótulo identifica a política viável de menor CTI; e o PSE
    aprende esse padrão para recomendar uma política e um ranking a novas
    séries. Cada etapa tem uma responsabilidade distinta.
  }
\end{frame}
