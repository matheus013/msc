% Slide 5 — Pergunta de pesquisa
\begin{frame}{Pergunta de pesquisa}
  \begin{quote}
    \Large É possível construir uma abordagem metodológica que
    \highlight{aprenda qual política de reposição é mais adequada} para cada
    combinação loja-produto, a partir de características operacionais da
    série, oferecendo uma alternativa empiricamente avaliada às políticas
    únicas aplicadas uniformemente à rede?
  \end{quote}
  \note{
    Essa é a questão que organiza toda a dissertação. Note que ela não
    pergunta ``qual é o melhor algoritmo de reposição''. Ela pergunta se é
    possível aprender, a partir de dados, qual política tende a funcionar
    melhor para cada série, usando as características operacionais
    observadas. Essa formulação já antecipa o papel do PSE como o
    componente que aprende essa relação.
  }
\end{frame}
